# 評論管理

### 新增評論

新增主題評論
                    
|URL|請求方法|參數|參數名稱|必填|型態|備註|權限|
|------------- | -------------|--------|-------|------|-----|-----|-----|
|theme |POST  |theme_id|文章/主題id|V|string|CREATE|
|                     |    |message |訊息|V|string||
