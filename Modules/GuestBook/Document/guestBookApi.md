|-------------------|------------------------|--------------|--------------|-----------------------------|
| <b>路徑</b>         |domain/guestbook/manager       |              |              |                      |
| <b>方法</b>               | POST          |              |              |          |
| <b>權限</b>               | -     |              |             | -         |
| <b>參數</b>               |                                    |              |              |        |
|                           | page    | 非必填(integer)     |         1     | 頁數          |
