| 項目                        | 內容                              | 類型         | 預設         | 說明        |
|-------------------|------------------------|--------------|--------------|-----------------------------|
| <b>路徑</b>         |domain/news/category/       |              |              |                      |
| <b>方法</b>               | POST          |              |              |          |
| <b>權限</b>               | -     |              |             | -         |
| <b>參數</b>               |                                    |              |              |        |
|                           | page    | 非必填(integer)     |         1     | 頁數          |
|                           | perpage   | 非必填(integer)           |   20     | 一頁數量            |
|                           | search   | 非必填(string)           |        | 關鍵字搜尋            |
